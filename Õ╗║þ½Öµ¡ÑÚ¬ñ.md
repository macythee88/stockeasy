# StockEasy PWA — 建站步骤指南
# 按顺序跟着做，每步约5–15分钟

---

## 第一步：注册 GitHub（约5分钟）

1. 打开 https://github.com
2. 点 "Sign up"，用 email 注册
3. 免费账号就够了
4. **完成后告诉我，我来帮你上传代码**

---

## 第二步：注册 Supabase（约10分钟）

1. 打开 https://supabase.com
2. 点 "Start your project"
3. 用 GitHub 账号登录（更方便）
4. 点 "New project"，填写：
   - Project name: `stockeasy`
   - Database password: 设一个强密码（记下来）
   - Region: **Singapore**（离马来西亚最近）
5. 等待约2分钟建立完成

### 建立数据库表格：
6. 进入 Supabase 项目 → 左边菜单 → "SQL Editor"
7. 点 "New query"
8. 把 `supabase/schema.sql` 文件的内容全部贴上去
9. 点 "Run"
10. 看到 "Success" 就完成了

### 建立照片存储空间：
11. 左边菜单 → "Storage"
12. 点 "New bucket"
13. Bucket name: `product-photos`
14. 勾选 "Public bucket"
15. 点 "Save"

### 复制 API 密钥：
16. 左边菜单 → "Settings" → "API"
17. 复制 "Project URL" 和 "anon public" key
18. 把 `.env.example` 改名为 `.env.local`
19. 填入这两个值

---

3. 完成后等我上传代码

## 第四步：上传代码到 GitHub（我来指导）

完成前三步后告诉我，我会给你具体指令。

## 第五步：在 Vercel 部署（约5分钟）

1. 在 Vercel → "New Project"
2. 选你的 GitHub repository
3. 在 "Environment Variables" 加入：
   - `VITE_SUPABASE_URL` = 你的 Supabase URL
   - `VITE_SUPABASE_ANON_KEY` = 你的 anon key
4. 点 "Deploy"
5. 约2分钟后得到网址如：`stockeasy.vercel.app`

---

## 第六步：加子域名 app.upin-global.com（约10分钟）

你的 DNS 在 WordPress.com，步骤如下：

1. 登录 https://wordpress.com
2. 左边菜单 → "Upgrades" → "Domains"
3. 点 "upin-global.com"
4. 点 "DNS Records" → "Manage"
5. 点 "Add a record"，填写：
   - Type: **CNAME**
   - Name: **app**
   - Value: **cname.vercel-dns.com**
   - TTL: 3600
6. 点 "Save"
7. 等待约30分钟生效

然后在 Vercel 里：
8. 进入你的项目 → "Settings" → "Domains"
9. 加入 `app.upin-global.com`
10. Vercel 会自动签发 SSL 证书

完成！打开 https://app.upin-global.com 就是你的 StockEasy 了。

1. 打开 https://app.upin-global.com
2. 底部分享按钮 → "Add to Home Screen"
